🦟 Price → Impact

This extension annotates prices on amazon.com with the equivalent impact in high-impact charity donations.

Today it ships with the Against Malaria Foundation. More charities, currencies, and sites are on the roadmap.

For the full converter (any currency, multiple charities), visit the project page.

Cost figures are GiveWell-derived approximations. See the source for details on which numbers, when they were captured, and how to refresh.
